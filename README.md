# Website-Screenshot-API
Fast screenshot rendering, no need to wait. You can try out our service for free, no need to register. We provide highly available service, with 99.9% uptime!

## Deployment

To deploy this application to Vercel, follow these steps:

1. **Install the Vercel CLI:**
   ```bash
   npm install -g vercel
   ```
2. **Deploy the application:**
   ```bash
   vercel
   ```
   Follow the prompts to link your project and deploy it.
